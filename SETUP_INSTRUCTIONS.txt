------------------------

4. Click "Go"

DATABASE STRUCTURE
------------------
- users: Farmer accounts with profile info
- mandis: 10 UP markets with distances
- crops: Crop catalog
- farmer_crops: Crops listed for sale
- price_data: Daily prices from each mandi
- sales_transactions: Completed sales history
- predictions: Price predictions
- alerts: Notifications

TROUBLESHOOTING
---------------
If login fails:
1. Check database is imported correctly
2. Check user exists in phpMyAdmin
3. Password must be at least 6 characters

If prices not showing:
1. Import the SQL file again
2. Check price_data table has records

If "Database Not Ready" error:
1. Make sure XAMPP MySQL is running
2. Check config/database.php credentials
